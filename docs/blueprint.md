# **App Name**: ApexFolio

## Core Features:

- Home Page: Hero section with name, role, value statement, and CTAs.
- Projects Showcase: Display project cards with screenshots, descriptions, and tech stacks.
- About Page: Short, human-focused page with developer background.
- Contact Form: Simple form with email submission.
- Skills Section: Showcase key skills with icons.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a sophisticated and attractive feel.
- Background color: Dark Charcoal (#212121) for a modern dark theme aesthetic.
- Accent color: Electric Purple (#BB86FC) to highlight important elements and CTAs with vibrancy.
- Font: 'Roboto', a modern sans-serif, for both headlines and body text to maintain a clean and readable look throughout the website.
- Plenty of whitespace to ensure a premium, uncluttered design and readability on dark backgrounds.
- Minimalistic, neon-style icons representing skills and technologies for visual appeal.
- Subtle glow effects on project cards and CTAs to enhance interactivity and provide a visually appealing experience.
